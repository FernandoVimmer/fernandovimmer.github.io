Projeto Voto Consciente

Abordamos este tema devido ao aumento das discussões políticas nas mídias sociais que são frequentes e pouco produtivos, pois os fatos discutidos nem sempre são relevantes para a escolha dos candidatos, também devido ao aumento significativo de notícias falsas relacionados a este assunto e que podem influenciar algumas pessoas que não buscam saber a veracidade da informação que foi obtida, e a falta de uma mídia mais interativa e de fácil acessibilidade para o público, pois algumas informações são díficeis de se encontrar.


Tela de login

Esta é a primeira tela do aplicativo, onde o usuário pode fazer o login caso seja cadastrado, recuperar a senha, ou criar um novo cadastro.



Tela de cadastro de novo usuário

Nessa tela o usuário pode criar um novo cadastro para acesso ao aplicativo.



Tela inicial

Essa é a tela inicial após o login ser realizado, permite o acesso as principais opções de consulta, notícias e perfil do usuário.



Escolha de Estado

Mapa do Brasil é exibido ao usuário para que possa consultar a partir de um determinado estado.



Confirmação do Estado

Estado selecionado pelo usuário é destacado na tela para confirmação de consulta.



Escolha de cargo político

Exibe as opções de cargos políticos disponiveis para consulta, ao selecionar o cargo exibe os políticos em mandato atualmente.



Lista de políticos

Nessa tela é exibido a lista de políticos para o cargo selecionado e sua avaliação mostrada ao lado em estrelas, sendo cinco o máximo e um o mínimo. Para visualizar a ficha do político basta seleciona-lo e clicar em pesquisar.



Dados do político

Ao selecionar o político que deseja consultar, é exibido a ficha dele com algumas informações básicas e abaixo disso abas específicas para consultar os projetos, processos, notícias, e todas outras informações disponiveis sobre ele. Também há a opção de "favoritar", para que possa selecionar esse candidato e ele aparecerá na tela de favoritos, assim poderá acessar a ficha dele facilmente e será notificado em caso de atualizações.



Lista de processos

Essa é a tela da lista de processos do político, mostrando um breve resumo sobre cada um, e ao clicar no símbolo "+" acessa a tela de detalhes sobre o processo. Abaixo é mostrado o total de processos.



Detalhes do processo

Nessa tela é exibido os detalhes do processo selecionado, com o status atual do processo.



Lista de projetos

A lista de projetos tem o mesmo modelo da lista de processos, exibindo resumidamente cada projeto e ao clicar no símbolo de expansão é possivel visualizar todas informações sobre o projeto e abaixo o total de projetos do político.



Detalhes do projeto

Ao selecionar um projeto específico é exibida essa tela onde mostra os detalhes desse projeto, e o status atual também.



Tela de perfil do usuário

Ao clicar em perfil é exibido seu perfil com as informações que inseriu no cadastro e suas informações de login e senha, onde é possível realizar alterações e deletar informações.



Tela de notícias

Nessa tela é mostrado as notícias recentes do político, onde pode clicar no símbolo de expansão para visualizar mais detalhes sobre a notícia selecionada.



Detalhes da notícia

Ao selcionar a notícia é exibida essa tela com todas as informações, referência, possibilitando aprovar ou reprovar aquela notícia, e comentar na aba "Comentários".



Tela de favoritos

Essa é a tela de Favoritos, que mostra os políticos que selecionou para acompanhar, os círculos amarelos indicam que há alguma atualização daquele político e quantas são. A opção de "favoritar" fica na ficha do político.



Comentários

Essa é a tela exibida após selecionar a opção Comentários em uma notícia, projeto, ou processo. Mostra o comentário de todos usuários e dá o espaço para inserir o seu comentário.



Feedback do Usuário

Ao selecionar a opção de feedback na tela inicial, é possível inserir um comentário como nessa tela, para enviar um feedback a equipe do aplicativo. Também mostra os feedbacks já realizados até o mesmo ser respondido.



Recuperação de senha

Essa é a tela de recuperação de senha caso o usuário tenha esquecido. Basta inserir o seu email, aguardar o código de recuperação e inseri-lo para gravar uma nova senha.
